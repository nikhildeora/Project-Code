# To Run Locally
## Clone the project
```bash
  git clone https://github.com/nikhildeora/zoomcar.git
```

Go to the project directory

```bash
  cd zoomcar

```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run start
```
