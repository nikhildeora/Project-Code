# To Run Locally
## Clone the project
```bash
  git clone https://github.com/nikhildeora/insidious-experience-328.git
```

Go to the project directory

```bash
  cd insidious-experience-328

```

Install dependencies

```bash
  npm install
```

Start the server


```bash
  npm run start
```
